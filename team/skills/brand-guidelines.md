## トーン&マナー
- **日本語**: 親しみやすく、でも大人っぽい。神戸の夜の雰囲気を大切に。
- **英語**: Casual but sophisticated. "Hidden gem", "local favorite" などの言葉を使う。
- 絵文字は控えめに。🏮 は定番でOK。

## ブランドワード
- 神戸 / 立ち飲み / 角打ち / 銘店 / 夜散歩
- Kobe / Tachinomi / Standing bar / Night walk / Local favorite

## アイコン・絵文字
- 🏮 — 居酒屋・立ち飲みの代表アイコン
- ⭐ — レーティング
- 📍 — 地図・場所

## 写真方針
- 暗め・ドラマチックな雰囲気が合う
- Google Places API の photo_reference を使用
- フォールバック: 🏮 + グラデーション背景
