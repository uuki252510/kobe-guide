## プロダクト概要

**Kobe Night Guide** は、神戸の立ち飲み・角打ち・居酒屋を外国人観光客にも使いやすい形で発信するWebアプリ。
AIチャットで店を提案し、インタラクティブマップで周辺店舗を探せる。

- **URL**: kobe-guide.vercel.app
- **スタック**: Next.js 15 / Supabase / Leaflet / Claude API
- **主要ターゲット**: 神戸在住外国人・インバウンド観光客・国内立ち飲みファン
- **掲載店舗数**: 約50店舗（拡大中）

## 開発ルール

1. **TypeScript必須** — any型禁止
2. **App Router構成** — pages/は使わない
3. **モバイルファースト** — タップターゲット44px以上
4. **ダーク×ゴールド** — #0E0E10 × #D4A842 がブランドカラー（マップUI）
5. **多言語配慮** — 英語表記を必ず併記
6. **Claude API使用** — チャット推薦はclaude-sonnet-4-6
